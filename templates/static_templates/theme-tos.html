Ce Code d'honneur n'a pas pour objet d'interdire les discussions à propos du contenu des cours. Au contraire, toute personne est autorisée voire encouragée à discuter du contenu de la formation avec les autres apprenants, dans la mesure ou ces échanges respectent le code d’honneur et les CGU.
